## 📖 插件简介

SkyBlock 是一款基于 **LiteLoaderBDS 2.5.0+** 开发的基岩版专用空岛插件，完美适配官方 BDS 服务端。

插件内置空岛核心玩法与进阶系统，支持多人联机、岛屿保护、权限管理、等级系统、挑战任务等全套功能，所有内容均可自由配置，轻松打造个性化空岛服。

# 空岛插件（skyblock）指令表

## 玩家指令（/is 或 /island）

| 命令                 | 别名      | 权限   | 描述                                               |
| -------------------- | --------- | ------ | -------------------------------------------------- |
| `/is`                | `/island` | 所有人 | 打开岛屿主菜单（若玩家无岛屿则自动进入创建流程）   |
| `/is create`         | -         | 所有人 | 打开岛屿创建菜单，选择蓝图生成新岛屿               |
| `/is delete`         | -         | 岛主   | 删除当前岛屿（需二次确认，执行后10秒传送回重生点） |
| `/is spawn`          | -         | 岛成员 | 传送至自己的岛屿出生点                             |
| `/is set spawn`      | -         | 岛主   | 将当前位置设为岛屿出生点（需在岛屿内）             |
| `/is set perms`      | -         | 岛主   | 打开岛屿权限设置表单                               |
| `/is invite`         | -         | 岛主   | 无参数时弹出在线玩家列表；带玩家名则直接邀请该玩家 |
| `/is kick <玩家名>`  | -         | 岛主   | 将指定成员踢出岛屿                                 |
| `/is help`           | -         | 所有人 | 查看岛屿帮助信息                                   |
| `/is setworld`       | -         | OP     | 打开世界权限设置表单（全局）                       |
| `/is level get`      | -         | 岛成员 | 计算并获取当前岛屿等级（需等级模块）               |
| `/is level setvalue` | -         | OP     | 设置手持方块的价值（需等级模块）                   |
| `/is level getvalue` | -         | 所有人 | 获取手持方块的价值（需等级模块）                   |
| `/is level top`      | -         | 所有人 | 查看岛屿等级排行榜（需等级模块）                   |

## 任务系统指令（/challenges）

| 命令                     | 权限   | 描述                                        |
| ------------------------ | ------ | ------------------------------------------- |
| `/challenges`            | 所有人 | 打开岛屿挑战任务菜单                        |
| `/challenges add <名称>` | OP     | 将手持物品保存为NBT奖励物品（用于任务奖励） |

## 控制台/调试指令

| 命令                                | 执行者    | 描述                                                 |
| ----------------------------------- | --------- | ---------------------------------------------------- |
| `/convertsky`                       | 控制台    | 将旧版空岛数据转换为新版格式（仅首次安装时可能需要） |
| `gamerule commandBlockOutput false` | 控制台/OP | 临时关闭命令方块输出，减少控制台刷屏                 |
| `gamerule commandBlockOutput true`  | 控制台/OP | 恢复命令方块输出                                     |

## 自定义命令用法示例

- **邀请玩家**：`/is invite` → 选择玩家 → 对方收到弹窗
- **接受邀请**：在弹窗中点击“接受”即可（无需指令）
- **删除岛屿**：`/is delete` → 输入 `/is delete` 确认 → 等待10秒后自动传送
- **查看任务**：`/challenges` → 选择等级 → 选择任务 → 查看详情并提交

## 注意事项

- 所有表单操作均通过点击按钮完成，无需手动输入复杂指令。
- 岛主权限：只有岛屿所有者才能执行踢人、设置权限、删除岛屿等操作。
- 岛屿成员上限、重置次数等受配置文件限制。
